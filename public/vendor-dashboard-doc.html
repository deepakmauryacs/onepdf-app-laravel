Vendor Dashboard Guide

Learn how to use all features of your OnePDF Vendor Dashboard

End-to-End “Happy Path”

  1. Create account → Pick plan → Verify email → auto-land on Dashboard.
  2. Upload PDFs (drag & drop or browse) → each file shows progress/state (PDF only, up to 50 MB).
  3. Manage files → generate secure share link / embed → copy link → (optional) open Details to set Title/Description.
  4. Set link permissions (Allow Download/Print, Require Email, Visibility) → Save.
  5. Share → later review Analytics (views, top documents, devices/geo).

Tip: Use Permissions to quickly toggle download/print without changing the link.

Dashboard

Snapshot of recent activity with quick actions.

  • Zero state: “Your library is empty. Drop PDFs here or click Upload.”
  • Recent uploads: last files with quick actions (Copy link / Details).
  • Counters: total documents, total views, downloads.

Files

Organize, share, and manage documents.

Upload Workflow

  • Drag & drop or Browse → upload starts automatically with a per-file row (name, size, progress).
  • Validation: PDF only; max size 50 MB. Errors appear inline.
  • On success: status “Ready • Get link” → primary action opens Details/Permissions.

Manage (Bulk)

  • Batch select rows → actions (Delete, Make Private, Get Links).
  • Row actions: Copy link, Open Details, Embed, Delete.
  • “Secure” pill shows when link is restricted (download/print off or private visibility).

Empty/Error States

  • Only PDF files are supported. Try again with a .pdf.
  • File exceeds 50 MB. Reduce size and retry.
  • Link revoked: “This link is inactive. Generate a new link or enable in Permissions.”

File Details & Permissions

Details

  • Editable: Title, Description, Tags.
  • System: size, upload date, link status (Secure).
  • Actions: Copy Link, Get Embed, Regenerate, Revoke.

Permissions

  • Allow download (toggle)
  • Allow print (toggle)
  • Require email (viewer must submit email)
  • Visibility: Public / Private / Off (revoked)

Save updates → status pill & mini-diff confirm (“Download disabled”).

Embed

  • iFrame/script snippets with size presets (Full / Card / Minimal).
  • Copy buttons with “Copied!” feedback.

Analytics

Visualize performance and drill into audience insights.

  • Overview: Views by day, Unique viewers, Total downloads; “Top documents”.
  • Per-Document: referrers, locations, device breakdown.
  • Time range: 7d / 30d / 90d / Custom.
  • No data: “No views in this range. Share your link to start seeing activity.”

Plan

  • Current plan + usage bars (Docs stored, Bandwidth, Team seats).
  • Upgrade/downgrade with feature comparison.

Notifications

  • System & account notifications (document views, system updates).
  • Preferences: choose which alerts & delivery channel (email/in-app).

Help Center

  • Contact Support: open ticket form; SLA message in confirmation.
  • Knowledge Base: search articles for embeds, permissions, analytics.

Password

Keyboard Shortcuts

  • Global search: Ctrl + K focuses the header search anywhere.
  • Files list: A Select all, Del Delete selected, L Copy link (first selected).

QA Checklist

  • Upload validator: MIME + extension + size, and server re-validates.
  • Link states: public, private, revoked — UI pill updates instantly.
  • Regenerate rotates token; old token becomes revoked.
  • Analytics counters exclude internal IPs (optional setting).
  • Bulk actions: optimistic UI with rollback + per-row toasts.
  • Copy buttons use navigator.clipboard.writeText with fallback.
