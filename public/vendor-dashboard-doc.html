Vendor Dashboard – How It Works

Dashboard Files Analytics Plan Notifications Help Center Profile Password

Dashboard

The dashboard gives vendors a quick overview of their recent activity. It lists the latest uploaded documents and displays quick statistics like total documents and recent downloads.

Files

All file management lives here.

  • List: Browse every file you have uploaded.
  • Upload: Add new PDF files using the upload form.
  • Manage: Rename files, generate sharing links and delete documents you no longer need.
  • Details: View file specific information and update metadata like title or description.
  • Embed: Generate embeddable code snippets to display documents on external sites.

Analytics

Analytics pages show how your documents perform.

  • Overview: Charts and counters summarizing views and downloads.
  • Documents: Performance table for each document.
  • Document Detail: Drill down into a single document’s metrics.

Plan

The plan page displays your current subscription and available upgrades. Vendors can switch plans directly from this page.

Notifications

Shows system and account notifications. Use this page to stay informed about important updates.

Help Center

Submit support requests and view existing tickets. The manage page lists past requests for quick reference.

Profile

Edit personal information like name and email. Changes here affect your account settings.

Password

Change your account password. Enter the current password and a new password to update.
